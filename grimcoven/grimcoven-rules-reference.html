CORE GAMEPLAY LOOP

Grimcoven is a cooperative boss battler where players control Hunters tasked with defeating a Griefbound entity. The game operates on a strict four-phase round structure. Each round begins with players rolling Hunter dice, which serve as the primary resource for activating Actions. Players spend dice to move, attack, and manage resources, collecting Lament from defeated enemies and map spaces. Lament grants powerful upgrades but increases Corruption, which triggers negative effects and risks permanent transformation. The round concludes with enemy activations, corruption checks, and board refreshes.

The game ends in victory when the Hunters successfully kill the Griefbound by defeating its Final Attack card. The game ends in defeat if the Event deck is exhausted while a new card must be drawn, or if a Hunter dies outside of the Final Act. Individual Griefbound scenarios may introduce additional win or loss conditions. An appropriate Epilogue is read after the scenario concludes.

SETUP
Scenario & Mode Selection

Select a Griefbound and its corresponding Scenario from the Hunt Sheet. Each Griefbound features five scenarios, with Scenario 1 being the easiest and Scenario 5 the most difficult. Consult the Hunt Book for the chosen Griefbound to access scenario-specific component lists, map layouts, and special rules. Determine player count scaling and apply any Game Mode or difficulty modifiers before beginning physical setup. The Hunt Book serves as the primary reference for component quantities and scenario-specific deviations from standard setup.

Map & Environment Deployment

Assemble the Map using the provided tiles and the diagram codes listed in the Hunt Book. Most maps include a special three-space Environment tile marked with a question mark, which is placed later. Place Lament Rift tokens face up in the designated spaces as indicated in the Hunt Book. Place a Portal model in every space marked with a Portal symbol. Distribute Lament tokens on the map based on player count: place one Lament in each even-numbered Lament Rift for two or solo players; add one to Rift 7 for three players; add one to Rifts 1 and 7 for four players; and place one in every Lament Rift for five players. Place one random Treasure token face down in each space marked with a Treasure symbol.

Draw Environment cards from the deck as listed in the Hunt Book. Terrain cards are drawn randomly for each terrain type present in the scenario, revealed, and placed next to the map to apply global terrain effects. Landmark cards are drawn for each marked space on the scenario map, revealed, and used to place corresponding models or tokens on the map before the card is placed adjacent to the map. Area cards instruct you to add a new three-space map tile to the board, after which the card is typically discarded unless a specific coded tile is required.

Enemy Initialization

Prepare the Event deck by gathering all event cards, selecting three Griefbound-specific events, and adding random events from other Griefbounds as specified in the Hunt Book. Shuffle the deck, place it next to the map, and position the Event Step I standee nearby. Select the Minion card specified in the Hunt Book and place it face up. Gather eight matching Minion models and place them in reserve. Spawn starting Minions by placing two models per player (maximum eight) on Lament Rifts starting from Rift 1 and proceeding in ascending numerical order. Position the Minion Step II standee near the Minion card.

Select the Elite enemy card specified in the Hunt Book and place it face up. Position the matching Elite model on the map unless the scenario instructs otherwise. Set the Elite level token to the number of players minus one. Position the Elite Step III standee nearby. Place the Griefbound board next to the map and insert the matching stat card. Set the Griefbound level token to the number of players minus one. Construct the Attack deck by drawing random cards from each tier according to the Hunt Book table, shuffling them, and placing them face down. Place the specified Final Attack card face down in the Attack deck slot, then cover it with the shuffled Attack deck. Fill the Attack track by placing the top card face up in the This Turn slot and the next card face up in the Next Turn slot. Place the specified Opportunity card face up in the This Turn slot beneath the Attack card. Position the Griefbound Step IV standee near the This Turn slot. Place the Time Track Step V standee on the start space of the Time Track. Position the Griefbound model on the map as directed.

Hunter Initialization

Each player selects a Hunter board in their preferred color and places it in front of them. Choose a Hunter tile, gather all corresponding Item and Evolution cards, and place the tile in the center of the board. Set the health dial to the maximum value. The player with the lowest maximum health takes the First Player token. Place the colored Assist marker in the Assist slot. Place one Corruption marker on the zero space of the Corruption track. Shuffle the Evolution deck and place it face down near the board. Select one Weapon and one Armor card as starting gear, place them face up, and gain the starting dice listed on those cards. Place the Hunter model on a designated starting space on the map. Apply any Hunter-specific setup instructions.

Final Setup Steps

Place all remaining dice and tokens in a common pool near the map. Shuffle the Corruption deck and place it face down next to the map. Place the Secret deck next to the map without shuffling it. Pass the First Player token to the appropriate player. Read the scenario Prologue. Each player rolls their dice to begin the game. Proceed to the Hunter Phase.

ROUND STRUCTURE
Hunter Phase Turn Order & Passing

Hunter turns resolve in clockwise order starting with the First Player. Each Hunter performs a turn by resolving exactly two Actions. After completing two Actions, play passes to the next Hunter in clockwise order. Hunters may continue taking turns as long as they have available dice and unspent Actions. If a Hunter cannot or chooses not to resolve more Actions, they must Pass. Upon passing, the Hunter discards all remaining dice from their active pool and is skipped for the remainder of the phase. The Hunter Phase continues until all players have passed. The last player who has not passed may take consecutive turns until they pass.

Dice & Action Resolution

Hunter dice are six-sided and come in four colors representing different skills. Offensive capabilities use the sword symbol. Speed and dexterity use the boot symbol. Defensive abilities use the shield symbol. Arcane prowess uses the magic symbol. To activate an Action, players place dice into matching slots. Slots with no symbol accept any die. Slots with a specific symbol require a matching die; a double symbol die may be used, but the secondary symbol is lost. Slots requiring two symbols may be filled by one double symbol die or two single symbol dice stacked together. Some Actions offer alternative costs, allowing players to choose between different symbol combinations. Once an Action is used, it cannot be activated again until refreshed.

Dice exist in three states: the active pool, action slots, or the discarded pool. Refreshing an Action discards the dice from its slots into the discarded pool, making the Action available again. Restoring dice moves them from the discarded pool back to the active pool after rerolling. Actions resolve fully from left to right and top to bottom before another Action can be taken. Discarding components listed in an effect is a mandatory cost. Free Actions require no dice but may have internal costs. Universal Marker Actions are free but limited to once per round; placing a marker activates them, and refreshing removes the marker. Action descriptors on cards or map spaces are free to declare but may require discarding components to resolve.

Basic Actions

Run costs any symbol and allows movement up to the Hunter movement value. Sprint costs a boot symbol and allows movement up to the movement value plus one, permitting travel through Portals. Tactic costs any symbol and grants one Tactic token, which may be immediately used to reroll any number of dice in the active pool. Cleanse costs a shield symbol and allows discarding up to two Status tokens. Assist costs any symbol and transfers the Assist marker to another Hunter, who places it in their active pool. The Assist marker functions as a die matching the symbol on the original owner's board.

Movement & Combat

Movement is restricted to adjacent spaces physically connected to the current space. Ending movement on a space containing a Lament token or Treasure allows the Hunter to collect it. Ending movement on a space with a Status token requires the Hunter to take it. Passing through a space does not trigger collection. Each space holds a maximum of four models, with the Griefbound counting as two. Entering a full space requires pushing one model to an adjacent space. Range is measured in spaces and indicated by minimum and maximum values. Line of Sight requires a straight, unobscured path; only obstacles and fog spaces block it. Map edges count as obstacles. Models on the same space share Line of Sight.

Enemy attacks deal a set amount of damage or use enemy dice. Sum dice results and apply damage or Status effects. Discard Defense tokens to reduce incoming damage by the token value; surplus defense is lost. Status effects cannot be blocked by Defense. Apply damage and Status before resolving other effects. Mark damage on the Hunter health dial. Hunters may push models in any order when entering full spaces.

Enemy Phase Event & AI Activation (Steps I–III)

At the start of the Enemy Phase, check for Critical Injury cards and apply their effects. Resolve Event cards in order: apply permanent effects, draw a new Event, place optional tokens on the map, and resolve the main effect. If the Event deck is empty when a draw is required, the scenario is lost. Permanent Events remain active until specified otherwise and resolve in the order they were drawn. Event effects contain a general portion and a conditional portion that only applies if the hunted Griefbound matches the card name. Mandatory costs must be paid to resolve optional secondary effects.

Activate Minions starting with the model closest to any Hunter. Each Minion resolves one activation from its card based on listed conditions. All Minions have one health, reward one Lament on death, hold one token, and feature two activation options plus a dice effect section. Undead Minions have a passive ability. Spawning Minions follows a strict sequence: count existing Minions, find the Lament Rift numbered one higher, and place models in ascending rift order until models run out. If a rift is full, push a model out. If components run out, spawn as many as possible.

Activate Elites starting with the closest model. Each Elite resolves one activation based on card conditions. Elite health varies by card and level token. Killing an Elite rewards Lament to the killer and other players. Elites hold one token and feature a dice effect section. Wounding an Elite to its health limit triggers a death check: discard tokens, remove the model, flip the card face down, and roll a die to determine if it is truly dead or temporarily retreated. Face-down Elites are considered off-map. Discard one marker from face-down Elite cards each phase; when the last marker is discarded, the Elite spawns but does not activate until the next phase.

Enemy movement targets the closest Hunter. Ties in distance break toward the Hunter with the highest Corruption. Further ties or multiple valid paths are decided by players. Enemies entering full spaces swap models in this order: companions, other Minions, Elites, Hunters.

Griefbound Step (Step IV)

Resolve the Griefbound Attack card in the This Turn slot from top to bottom. Sum attack dice results with stat card modifiers and attack card dice effects. After resolution, discard the This Turn card to the discard pile, moving any tokens with it. Slide the Next Turn card to the This Turn slot, preserving its tokens. Refill the Attack track by drawing new cards to fill empty slots. If the Final Attack card becomes visible, reshuffle the discard pile to cover it and continue refilling. If the discard pile cannot cover the Final Attack card, trigger the Final Act.

Killing an Attack card requires dealing damage equal to or greater than its health value. All players gain Lament rewards. The killer places the card under their board as a trophy. If the killed card was in the This Turn slot, the killer rolls a die to resolve the uncovered Opportunity card. Opportunity cards provide bonuses or track refill instructions. Some results stagger the Griefbound, leaving the This Turn slot empty for the next phase; the track refills at the end of Step IV. Refilling the track always moves the Next Turn card forward and draws new cards. If the Final Attack is uncovered during refill, check for Final Act triggers.

The Final Act triggers when the Attack deck is depleted and two or fewer Attack cards remain in play and discard combined. Remove remaining Attack cards and Secret 2 cards. Reveal the Final Attack card in the This Turn slot. Level up the Griefbound and all Elites. If triggered during the Hunter Phase, immediately end the phase and proceed to the Enemy Phase. During the Final Act, Hunter death no longer ends the scenario. Defeating the Final Attack requires a die roll; success wins the scenario, failure removes damage tokens and lowers the Griefbound level. If the level cannot lower further, the scenario is won.

Advance the Time Track standee one space and resolve the entered space. Spawn spaces add two Minions. Stage spaces draw and resolve a Stage card, which may level up the Griefbound and Elites or add new effects. Special spaces resolve the Griefbound stat tile special effect.

Lament Phase

Each Hunter draws and resolves one Corruption card starting from the First Player. The top section resolves Hunger effects; skip this section if the Hunter has no Hunger. The bottom section resolves effects matching the current Corruption level. Place the resolved card on the bottom of the deck. Reshuffle if instructed. Corruption measures Lament abuse; higher levels trigger worse effects. Reaching level 15 flips the Hunter tile to the corrupted side. Losing Corruption moves the marker backward without losing evolutions or flipping the tile back. Re-entering a lock space allows evolution again.

Hunters spend Lament one token at a time to advance Corruption by one or discard one Hunger. Advancing to a lock space triggers an Evolution. Reaching level 15 flips the tile and resolves immediate effects. All remaining Lament is discarded. During Evolution, draw two cards, spend Lament to redraw, keep one card, or gain a book icon if none are kept. Unchosen cards go to the bottom of the deck. After spending, players may optionally advance Corruption to the next lock space without resolving the evolution. Final Perdition triggers via Secret cards at high Corruption, transforming the Hunter into a Pretender.

Refresh Phase

Pass the First Player token clockwise. Return used Environment cards to their proper locations. Return all Assist markers to their owners. Discard all frost tokens from cards. Gather all dice from action slots and the discarded pool, reroll them, and place them in the active pool. Hunters may spend magic symbols to discard an equal number of Status tokens before resolving negative effects. Place spent dice in the discarded pool. Resolve all Status effects. Burn tokens deal unblockable damage during this step and persist. Stun tokens block specific Actions and persist until removed. Curse tokens trigger Secret 1 cards; discarding the last curse returns the card to the deck. Hunger is not a Status and cannot be cleansed. If component pools are empty, skip the missing token effect and deal one damage instead.

SPECIAL RULES & EDGE CASES

Map spaces hold a maximum of four models. The Griefbound counts as two models. Entering a full space requires pushing a model to an adjacent space. Hunters choose which model to push. Enemies push in this order: companions, Minions, Elites, Hunters. Griefbound and environment models cannot be pushed. Terrain spaces share colors and apply global effects. Deep water spaces cannot be entered but allow range and line of sight measurement. Fog spaces block line of sight across borders but not within the same space. Obstacle spaces cannot be entered and block range and line of sight. Lament Rifts are numbered spaces used for spawning and placement; they do not override terrain effects.

Tokens on the map are ignored by enemies unless specified. Hunters collect positive tokens like Treasures and Lament when ending actions on those spaces. Treasures reveal random rewards, always including a heal option, and are removed after use. Negative tokens like Status and Hunger must be taken when ending actions on those spaces. Secret cards are drawn by number without shuffling the deck. Multiple cards may share a number; choose randomly. Secret 1 cards handle curse effects. Secret 2 cards handle critical injuries. Companions are gained by drawing a random card, placing it near the board, and positioning the model on the Hunter space. Companions are never targeted by enemies and are pushed first when spaces fill.

When a Hunter reaches zero health, heal one, discard all Status and Hunger tokens, and draw a Secret 2 card. The Hunter is ignored by all enemies and negative effects until the next event step. Complete the quest on the card before that step to heal and remove the card. Failure results in Hunter death and scenario loss. If no Secret 2 cards remain, the Hunter dies instantly. This rule changes during the Final Act.

GAME MODES & VARIANTS

All modes and difficulty modifiers are compatible. Players may combine any number of options.

Solo Mode utilizes a Deputy representing an unchosen Hunter. The Deputy uses a board with a deputy action tile covering basic actions. Place the chosen deputy card face up, three universal markers nearby, and the shuffled deputy evolution deck. Position the deputy model on the map. The Deputy resolves turns using universal markers instead of dice. All deputy action slots are blank. One marker activates one action. Refreshing discards a marker; restoring moves a marker from discarded to active. The Deputy and Hunter may assist each other. The Deputy cannot use markers for effects requiring specific symbols. The Deputy gains Lament instead of dice from rewards. During refresh, return all markers to the board. During evolution, the Deputy may spend Lament to discard Hunger, advance Corruption, gain a marker, or draw an evolution card. The Deputy counts as a Hunter for all effects. Deputy death ends the scenario.

Rapid Hunt accelerates gameplay. Remove random Attack cards from the deck. Resolve two Stage cards, including level ups. Set the Time Track to the last stage space without resolving it. Discard all Lament from the map. Discard the top three Event cards without resolving them. Set each Hunter Corruption to seven. Resolve evolution three times per Hunter. Apply Griefbound-specific modifications: Undead General reveals two banner cards. Swarm Mistress places two additional swarm tiles near hunters. Convergence grants three burns per hunter. Tormentor deals damage equal to scenario number. Graveyard Dragon grants two curses per hunter. Devourer grants two hunger per hunter. Pretender resolves special twice and gains two tokens per defense slot.

Difficulty Customization allows easy or hard modifiers. Easy options include resolving one evolution per hunter, gaining two tactic tokens, lowering Griefbound or Elite levels by one, switching to non-named elites, using convicted minions, adding two random events, or performing additional map setup. Hard options include increasing Griefbound or Elite levels by one, switching to named elites, using undead minions, or discarding the top event card.

PRETENDERS (CORRUPTED BOSSES)

Pretenders are corrupted Hunter variants that become additional Griefbound enemies. Unlock them by triggering Final Perdition. Select a random scenario and follow standard setup using the Pretender components. Place the Pretender model on a Griefbound base. Construct the Attack deck using seven cards for solo/two players, eight for three, nine for four, or all cards for five. Players cannot control the Hunter version of an active Pretender.

Pretender Duality uses light and dark tokens to empower attacks. Gain light places a token on the stat card. Light damage resolves additional effects if enough tokens exist, then discards used tokens.

Pretender Gunslinger uses a bullet track on the stat tile. Forge a bullet places a token in the rightmost empty slot. Bullets are used right to left in the special effect to increase damage or range. All bullets are used or none. Tokens discard after use.

Pretender Keeper collects craving tokens. At the start of Step IV, if five craving tokens exist, resolve the This Turn card twice and discard all craving. This triggers once per phase. Do not discard if no card is in the This Turn slot.

Pretender Technomancer uses enhancement markers. Enhance places a marker on Attack cards starting from This Turn. Cards with markers resolve enhanced effects, roll an extra die during attacks, and resolve marked dice effects. Each card holds one marker.

Pretender Death Jester gathers revealed fate cards in an unlimited sequence. Attack cards or dice effects check the sequence for specific ranks. Resolving a rank triggers additional effects. Used cards return to the deck at the end of Step IV. The Hanged Man resolves instantly per stat card instructions.

Pretender Mannequin collects active personality tokens on the stat card. Specific personalities boost Attack cards. Activate random personality places a random token on the stat card. Hunters may obtain string cards, but the Pretender counts as an enemy for all string effects.
